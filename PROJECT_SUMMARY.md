## 📋 **Setup Instructions**

### **Quick Start**
1. **Install dependencies**: `npm install`
2. **Set up database**: `npm run db:push`
3. **Seed database**: `npm run db:seed`
4. **Start development**: `npm run dev`
5. **Access**: http://localhost:3000
